# 📸 Arianna — Wedding & Portrait Photographer Portfolio

Элегантное портфолио фотографа с динамическим фоновым видео и оптимизированной галереей.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Arianna/)

---

### 🌟 Особенности проекта:
* **Hero-видео:** Полноэкранный видео-фон с атрибутами оптимизации (`muted`, `playsinline`, `poster`).
* **Оптимизация изображений:** Использование атрибута `loading="lazy"` для ускорения загрузки страницы и экономии трафика.
* **SEO-подготовка:** Настроены мета-теги Open Graph для красивого отображения ссылок в социальных сетях.
* **Адаптивная верстка:** Удобная навигация и сетка галереи, которая подстраивается под мобильные устройства.

---

### 🛠 Стек технологий:
* **HTML5:** Семантическая разметка (figure/figcaption, article, footer).
* **CSS3:** Flexbox-сетки, стилизация видео-контейнеров и кастомные шрифты Google Fonts.
* **Video API:** Интеграция фонового видео.
